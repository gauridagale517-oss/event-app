Event Finder App

A Flutter-based mobile application that allows users to discover and explore nearby events such as concerts, sports matches, and social gatherings.

---

## 🚀 Features

- 🔍 Browse events from API
- 📄 View detailed event information
- 🔎 Search events by title and location
- 🔄 Pull-to-refresh functionality
- 📱 Clean and responsive UI
- 🔗 API integration using GET request

---

## 🛠️ Tech Stack

- **Frontend:** Flutter (Dart)
- **API Handling:** http package
- **Mock API:** Beeceptor
